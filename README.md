# Akan Name Generator

A simple web application that calculates the day of the week a user was born and assigns them an Akan name based on Ghanaian tradition. Users enter their birth date and gender to discover their Akan name.

## Project Description

The Akan people of Ghana traditionally name their children according to the day of the week they were born. Each day has a corresponding name for boys and girls. This web app calculates your birthday and displays your Akan name based on your gender.

This project allows users to:

- Enter the day, month, and year of birth separately
- Select their gender (male/female)
- Get their Akan name along with the day of the week they were born
- Receive alerts if invalid data is entered
